## âœ¨ Features

- **Embeddable Chat Widget**: Powerful AI chat widget built with AI SDK, fully customizable and easy to style with great developer experience.

- **One-Line Integration**: Add to any platform with a single script tag - works with Docusaurus, Fumadocs, Next.js, HTML, Webflow, and practically any website.

- **One-Click Crawls**: Automated web crawling with scheduling support and CI/CD integration to keep your knowledge base up-to-date.

- **Analytics Console**: Comprehensive dashboard with conversation analytics, usage metrics, and detailed conversation history.

- **Model Flexibility**: Integrate any embedding model and LLM provider - swap models without changing your code.

- **Knowledge Sources** *(coming soon)*: Connect APIs, Slack, Discord, and more as knowledge sources for your AI assistant.

- **Simple Self-Hosting**: Pure TypeScript stack on PostgreSQL with minimal moving parts - scalable, robust, and built on popular, battle-tested technology.
